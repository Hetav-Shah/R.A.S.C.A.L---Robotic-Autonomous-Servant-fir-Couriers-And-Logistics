# 🤖 RASCAL  
### Robotic Autonomous Servant for Courier & Logistics

## 🚀 Overview
RASCAL is an autonomous mobile robot designed for last-mile delivery applications in courier and logistics environments.

The goal of this project is to develop a robot capable of navigating real-world environments, avoiding obstacles, and delivering packages efficiently with minimal human intervention.

---

## 🧠 Problem Statement
Traditional delivery systems rely heavily on human effort and lack adaptability in constrained or dynamic environments.  

RASCAL aims to address this by building a scalable robotic platform that can:
- Navigate autonomously  
- Handle real-world uncertainties  
- Deliver items with precision  

---

## ⚙️ Prototype 1 (Completed)
The first prototype focuses on building a **functional autonomous delivery system** with basic navigation capabilities.

### 🔹 Key Features
- Autonomous navigation using GPS  
- Mobile-controlled operation  
- Basic obstacle avoidance  
- Real-world delivery demonstration  

### 🔹 Tech Stack
- Arduino + GPS module  
- C++ for control logic  
- Mobile app integration  
- Mechanical design in SolidWorks  

### 🔹 Outcome
Successfully demonstrated:
- Autonomous movement to target locations  
- Remote-controlled delivery via mobile interface  
- Stable mechanical design for navigation  

---

## 🚧 Prototype 2 (In Progress)
The second prototype focuses on **intelligence, perception, and robustness**.

### 🔹 Upgrades
- Raspberry Pi for onboard computation  
- LiDAR for environment sensing  
- ROS integration for system architecture  
- Real-time localization and mapping  

### 🔹 Goals
- Accurate self-localization  
- Dynamic obstacle avoidance  
- Environment mapping (SLAM)  
- Improved navigation efficiency
